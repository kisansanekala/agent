Langkah selanjutnya

Ikuti urutan ini biar Lapak.AI siap jawab chat pelanggan di halaman Toko Publik.

  1. Isi Profil Toko — nama, deskripsi, kontak.
  2. Upload foto & detail di Produk.
  3. Lengkapi FAQ buat pertanyaan yang sering muncul.
  4. Masukkan API key Groq di Pengaturan AI.
  5. Buka Lihat Toko Publik buat lihat hasilnya sebagai pelanggan.

Pengaturan AI

Sambungkan Groq API supaya Lapak.AI bisa menjawab chat pelanggan secara otomatis.

Ambil kunci gratis di console.groq.com/keys. Kunci disimpan hanya di browser ini.

Model Llama lama (3.3 70B / 3.1 8B) sudah dipensiunkan Groq per Juni 2026, jadi dipakai penggantinya.
