Langkah selanjutnya

Ikuti urutan ini biar Lapak.AI siap jawab chat pelanggan.

  1. Isi Profil Toko — nama, deskripsi, kontak.
  2. Tambahkan produk & FAQ di Basis Pengetahuan.
  3. Masukkan API key Groq di Pengaturan AI.
  4. Coba ngobrol di Uji Coba Chat.

Pengaturan AI

Sambungkan Groq API supaya Lapak.AI bisa menjawab chat secara otomatis.

Ambil kunci gratis di console.groq.com/keys. Kunci disimpan hanya di browser ini, tidak dikirim ke server manapun selain Groq.

Model Llama lama (3.3 70B / 3.1 8B) sudah dipensiunkan Groq per Juni 2026, jadi dipakai penggantinya.
